# Changelog

All notable changes are documented here automatically by [semantic-release](https://github.com/semantic-release/semantic-release).

Commit messages follow the [Conventional Commits](https://www.conventionalcommits.org/) spec:

| Prefix | Release type | Example |
|--------|-------------|---------|
| `feat:` | minor | `feat: add reverb tips to vocals page` |
| `fix:` | patch | `fix: tooltip position off-screen on mobile` |
| `perf:` | patch | `perf: reduce glossary re-render on search` |
| `BREAKING CHANGE:` | major | `feat!: redesign filter API` |
| `chore:`, `docs:`, `style:`, `refactor:`, `test:` | no release | — |
